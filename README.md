# Blumenwunder App

Webbasierte Warenwirtschafts-App für Blumenwunder Sassen.

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS
- Supabase Client
- React Router

## Setup

1. Abhängigkeiten installieren:
   ```bash
   npm install
   ```

2. Environment Variables setzen:
   Kopiere `.env.example` zu `.env` und fülle die Supabase Keys aus.

3. Starten:
   ```bash
   npm run dev
   ```

## Projektstruktur
- `/src/components`: UI-Komponenten (Buttons, Inputs, Cards)
- `/src/pages`: Hauptseiten (Dashboard, Einkauf, Verkauf)
- `/src/lib`: Supabase Konfiguration und Helper
- `/src/types`: TypeScript Interfaces
